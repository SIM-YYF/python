### 字符串的输出

```
    name = '新锐'
    print('--------------------------------------------------')
    print("姓名：%s"%name)
    print('--------------------------------------------------')
```

### 字符串的输入

```
    userName = input('请输入用户名:')
    print("用户名为：%s"%userName)
```

注意：input获取的数据，都以字符串的方式进行保存，即使输入的是数字，那么也是以字符串方式保存
