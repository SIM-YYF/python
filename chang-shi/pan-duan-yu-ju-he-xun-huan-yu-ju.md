### if-elif-else 判断语句

```
 if 条件:
      执行语句
       ...
 elif 条件:
       执行语句
       ...
 else:
       执行语句
       ...
```

### while循环语句

如：

```
i = 1
sum = 0
while i<=100:
    if i%2 == 0:
        sum = sum + i
    i+=1

print("1~100的累积和为:%d"%sum)
```
