## &lt;1&gt; 定义带有参数的函数 {#定义带有参数的函数}

```
    def add2num(a, b):
        c = a+b
        print c
```

## &lt;2&gt; 调用带有参数的函数 {#调用带有参数的函数}

```
    def add2num(a, b):
        c = a+b
        print c

    add2num(11, 22) #调用带有参数的函数时，需要在小括号中，传递数据
```

## 注意 {#小总结}

* 定义时小括号中的参数，用来接收参数用的，称为 “形参”
* 调用时小括号中的参数，用来传递给函数用的，称为 “实参”
